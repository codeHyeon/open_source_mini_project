# Memo App

A simple memo application built with Flask.

## Features
- View memo list
- Add memo
- Delete memo

## Requirements
- Python 3
- Flask

## Installation

Install dependencies:
pip install -r requirements.txt

## Run

python app.py

Open browser:
http://127.0.0.1:5000

## Pages

- Home : View memo list
- Add Memo : Create a memo
- Delete Memo : Remove a memo
